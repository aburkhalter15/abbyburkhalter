Wayfinder Kiosk

Developing a Wayfinder kiosk was an internship opportunity that I had with working for UITS at Indiana Univeristy Kokomo. The wayfinder kiosk will be used by visitors to find people, places, or accessibility tools on campus. While the HTML and CSS are the framework that holds the wayfinder together, I used jQuery heavily for show and hide functions. The homescreen consists of three choices. A user may choose to search for people, places, or accessibility. If a user is searching for a person, they may choose people and they are sent to a list of alphabet letters. The user is told to choose the first letter of the last name of the person they are looking for. jQuery is used to show everyone who's last name begins with the chosen letter. When the user selects a name from the list presented, jQuery is used to show that person's contact information. This part was slightly more difficult because instead of simply showing something that was hidden, I pulled in a section from a completely different page. PHP is used to pull the persons that are being searched contact information from a database.

Project Details

  • HTML
  • CSS
  • jQuery
    • Show and hide functions
  • PHP

IU Kokomo About page

During my time as a web technician on the IU Kokomo web team, I was given many projects where the goal was to take a designed web page and turn it into HTML. For the IU Kokomo about page, I had to turn a brochure into HTML. I took each row and relatively positioned them one on top of the other. Then I absolutely positioned the items inside of the rows. I followed the spacing around the text in the design very closely when I positioned each item, which is why I used absolute positioning on the items within the relatively positioned containers. I provided alternative text when it was needed for the images to make the page accessible. I also modified the design of the web page for an ipad version and a mobile version. In the responsive versions, I got rid of unnecessary images.

Project Details

  • Each row a relative positioned container
  • Absolute positioning on items within containers

Javascript Course

I picked up very basic Javascript skills from a Javascript course I took at IU Kokomo. For this particular function, a user takes a quiz to see if they know what kind of dog is displayed in an image. I declared 5 variables, one for the quiz, two for each correct answer, and two for the outputs. I used if statements to determine the output of the function. If the user chooses akita in the first question, then the output is correct. Else if they choose anything else, the output is wrong. Same concept for the second question, only the correct answer for that one is beagle.
